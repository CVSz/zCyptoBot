# ZCyptoBot Growth Engine Blueprint (Referral + Viral + AI Automation)

## 1) North Star and Growth Math
- **North Star Metric (NSM):** Net Revenue Retention-adjusted ARR growth.
- **Leading Indicators:**
  - Qualified signups/week
  - Activation rate (Aha moment < 24h)
  - K-factor (invites × invite conversion)
  - Sales-assisted close rate
  - Payback period (months)

## 2) Full-Funnel Growth System
1. **Acquire**: content, paid, partners, community.
2. **Activate**: in-product onboarding + one-click exchange sandbox.
3. **Monetize**: usage-based + platform + enterprise add-ons.
4. **Refer**: dual-sided referrals, analyst affiliate program.
5. **Expand**: multi-desk rollout, advanced AIops modules.

## 3) Referral System Design
### Program structure
- **Dual-sided rewards:**
  - Referrer: 15% revenue share for 12 months or equivalent platform credits.
  - Referred account: onboarding fee waiver + 2-month discounted platform fee.
- **Partner tiers:** Bronze/Silver/Gold based on annual influenced ARR.
- **Fraud controls:** device fingerprinting, payout hold period, anomaly scoring.

### Product mechanics
- Unique referral links + invite codes.
- Team invites from admin console.
- Attribution window: 90 days, last non-direct touch override for partner links.
- Monthly payout reporting via finance webhook exports.

## 4) Viral Loops (Built-In)
### Loop A: Insight Sharing Loop
User configures alpha dashboard → shares benchmark snapshot → recipient opens interactive view → prompted to run own sandbox.

### Loop B: Collaboration Loop
Portfolio manager invites risk/compliance peers → teammates customize policy packs → each new teammate invites adjacent functions.

### Loop C: API/Template Loop
User publishes strategy template to private marketplace → peers fork + improve → forks require account creation.

## 5) AI Marketing Automation Stack
### Core agents
- **ICP Scorer Agent:** enriches inbound accounts, routes by fit + urgency.
- **Journey Orchestrator Agent:** launches multi-step email/ads/webinar sequences.
- **Creative Generator Agent:** drafts vertical-specific messaging and LP variants.
- **RevOps Optimizer Agent:** reallocates spend by CAC:LTV and conversion lift.

### Data model
- Source events: product telemetry, CRM, ad platforms, email, webinars.
- Unified entities: account, contact, opportunity, campaign, touchpoint.
- Model outputs: fit score, intent score, churn risk, expansion propensity.

### Automation playbooks
- **PQL acceleration:** if PQL score > threshold and no meeting, trigger SDR sequence.
- **Trial rescue:** if activation lag > 3 days, launch guided workflow + concierge CTA.
- **Expansion trigger:** if usage at >80% quota, push seat expansion/enterprise add-on offer.

## 6) 90-Day Rollout Plan
### Days 0–30
- Instrument funnel + data contracts.
- Launch referral MVP in app + CRM attribution.
- Ship one viral artifact (shareable performance snapshot).

### Days 31–60
- Launch AI scoring + lifecycle orchestration.
- Introduce partner tiering + monthly leaderboard.
- Deploy 3 persona-specific onboarding flows.

### Days 61–90
- Run growth experiments (price tests, LP variants, loop multipliers).
- Add expansion automation and CSM handoff playbooks.
- Formalize quarterly growth operating cadence.

## 7) KPI Targets (Year 1)
- Activation: 38% → 58%
- Referral contribution to new ARR: 6% → 22%
- K-factor: 0.18 → 0.42
- CAC payback: 20 months → 11 months
- NRR: 112% → 131%

## 8) Implementation Notes
- Keep all incentives compliant with jurisdictional and brokerage partner rules.
- Require role-based permissions for referral payouts and growth experiment toggles.
- Add holdout groups to measure true incrementality of automation and loops.
