# Security Policy

## Access Control
- RBAC enforced (least privilege)
- OIDC SSO required

## Data Protection
- Encryption at rest (AES-256)
- Encryption in transit (TLS 1.3)

## Audit Logging
- Immutable logs (append-only)
- Retention: 365 days

## Incident Response
- SLA: <1 hour detection
- Full RCA required
